### 2.除数表

**91：一个数除以91 等于 这个数错位相加**

&nbsp;&nbsp;&nbsp;&nbsp;91对应的是$\frac{1}{11}$，若$\frac{a}{91}$ = $\frac{a}{1/11}$ = a*11，,a乘以11和a乘以1.1有效数字是一样的，即a+0.1a,所以看到91做分母就要想到错位相加

**111:一个数除以111，等于 这个数错位相减**

&nbsp;&nbsp;&nbsp;&nbsp;111对应的是$\frac{1}{9}$，$\frac{A}{111}$ = A*9 = A * （10-1）相当于A - 0.1A,即错位相减

**125:一个数除以125，等于 这个数乘以8**

&emsp;125对应的是$\frac{1}{8}$，457 / 125 =457 * 8=3656

**133：一个数除以133，等于这个数减去它本身的1/4**

&emsp;$\frac{1}{3}$ = 33%，133相当于$1+\frac{1}{3}$ = $\frac{4}{3}$，$\frac{457}{133}$ = $\frac{457}{4/3}$ = 457 *$\frac{3}{4}$，相当于457减去它本身的1/4，即457-457/4=457-114=343

**143：一个数除以143，等于这个数乘以7**

&emsp;143对应的是$\frac{1}{7}$，$\frac{A}{143}$ = $\frac{A}{1/7}$ = A *7

**150:一个数除以150，等于这个数减去它本身的1/3**

&emsp;150对应的是$\frac{3}{2}$，$\frac{A}{3/2}$ = A*$\frac{2}{3}$ = A -$\frac{1}{3}A$

**167:一个数除以167，等于这个数乘以6**

&emsp;167对应的是$\frac{1}{6}$，$\frac{A}{1/6}$ = A*6

**222：一个数除以222，等于这个数除以2再错位相减**

&emsp;222=111*2，$\frac{457}{222}$ =$\frac{228}{111}$=228-23=205

**250：一个数除以250，等于这个数乘以4**

&emsp;250对应的是$\frac{1}{4}$，$\frac{457}{250}$ = 457 * 4 =1828

**270：一个数除以270且分子是整数，等于这个数乘以37**

&emsp;270对应的是$\frac{1}{37}$，$\frac{400}{270}$ = $\frac{400}{1/37}$=400 * 37 = 148

**333:一个数除以333，等于这个数乘以3**

&emsp;333对应的是$\frac{1}{3}$,$\frac{A}{1/3}$ = A *3

**370：一个数除以370且分子是整数，等于这个数乘以2.7**

&emsp;370对应的是$\frac{1}{2.7}$，$\frac{400}{370}$=400*2.7=1080

**444：一个数除以444，等于这个数除以4再错位相减**

&emsp;$\frac{789}{444}$ = $\frac{197}{111}$ =197 - 19.7 = 177

**500：一个数除以5，等于这个数乘以2**

&emsp;$\frac{A}{500}$ = $\frac{A}{5}$ = $\frac{A}{1/2}$=A*2

**555:一个数除以555，等于这个数乘以2再错位相减**

&emsp;$\frac{A}{555}$=$\frac{A}{5*111}$=$\frac{A*2}{111}$这个结果再错位相减

**666：一个数除以666，等于这个数加上它本身的一半**

&emsp;666对应的是$\frac{2}{3}$，$\frac{A}{2/3}$=A * $\frac{3}{2}$ = A +$\frac{1}{2}A$

**750:一个数除以750，等于这个数加上它本身的$\frac{1}{3}$**

&emsp;750对应的是$\frac{3}{4}$，$\frac{A}{3/4}$ =A * $\frac{4}{3}$ =A + $\frac{1}{3}A$

**857:一个数除以857，等于这个数加上它本身的$\frac{1}{6}$**

&emsp;857距离1000有143，而143对应$\frac{1}{7}$，那么857就是$\frac{6}{7}$，$\frac{A}{6/7}$ = A * $\frac{7}{6}$ =A + $\frac{1}{6}A$
